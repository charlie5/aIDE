# aIDE
A widget-based Ada IDE.

### Dependencies

- GtkAda
- Florist
- aShell ~ https://github.com/charlie5/aShell.git


### Build

- $ cd aIDE/applet/aide
- $ gprbuild -P aide.gpr

### Test

- $ cd test
- $ ../launch_aide

- Open the 'builder' expander (bottom left).
- Click on 'build' button.
- Close aIDE.

- $ ./launch_hello
